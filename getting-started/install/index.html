  • Windows

To install Flojoy on Mac & Linux:

  1. Install Redis, an open-source, in-memory data store used by Flojoy:
brew install redis

(Redis installation reference)

  1. Clone the repo with --recursive argument as follows:
git clone --recursive https://github.com/flojoy-io/studio.git
  1. Navigate into the project root:
cd studio
  1. We highly recommend creating a Python virtual environment inside the studio folder by running the command:
python3.10 -m venv venv
  1. Finally, run:
bash flojoy -v venv

To install Flojoy on Windows:

  1. Install Memurai, an alternative to Redis for Windows

  2. Update your PowerShell to at least >=v7.0.0

  3. Clone the repo with --recursive argument as follows:

git clone --recursive https://github.com/flojoy-io/studio.git
  1. Navigate into the project root:
cd studio
  1. We highly recommend creating a Python virtual environment inside the studio folder by running the command:
python3.10 -m venv env
  1. Finally, run:
.\flojoy -v env
Optional flags
  • -v provides the path to the Python virtual environment
  • -r shuts down the existing redis server and spin up a fresh one (For Mac and Linux)
  • -n skips installing the Node packages in package.json
  • -p skips installing the Python packages in requirements.txt
  • -P runs the backend server on a specific port
  • -s To **NOT** update submodules
  • -S To **NOT** enable Sentry
  • -T To enable Telemetry
  • -d To enable debug mode for backend
