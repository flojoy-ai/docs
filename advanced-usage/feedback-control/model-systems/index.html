Modelling real life systems

Modelling real-life systems​

A first order system​

In order to understand how we might simulate a realistic system to control with our active feedback controller, we first must understand what the 'order' of systems are. Many systems have very complicated relationships between what you input into them, and what they output. If this relationship, the transfer function x(ω)x(\omega)x(ω) only depends on first-order time derivatives of the input signal, the system is called 'first-order', and the output of such a system is a recursive or infinite impulse response filter (you can read more about these here).

In these systems, now going from a continuous-time system output to discrete time system output y(t)→y(tn)≡yny(t)\rightarrow y(t_n)\equiv y_ny(t)→y(tn​)≡yn​, you can show that the output of this filter will be:

yn=a0xn+a1xn−1+⋯+b1yn+b2yn−2+⋯y_n = a_0x_n + a_1x_{n-1} + \cdots + b_1y_n + b_2y_{n-2}+\cdotsyn​=a0​xn​+a1​xn−1​+⋯+b1​yn​+b2​yn−2​+⋯

where the aaa's and bbb's are called the recursion coefficients, defining the weights of each of its previous inputs and outputs. A simple low-pass filter (aj=0∀j≠0a_j=0\forall j\neq 0aj​=0∀j=0, bk=0∀k≠1b_k=0\forall k\neq 1bk​=0∀k=1) will then have the following transfer function:

x(ω)=a01−b1ωx(\omega) = \frac{a_0}{1-b_1\omega}x(ω)=1−b1​ωa0​​

The rise and decay times of a simple low-pass filter (ie RC circuit) follow a simple exponential form with the discrete number of samples (and therefore the time-constant of this filter) passed to the filter x=exp⁡{−1/d}x=\exp\{-1/d\}x=exp{−1/d}, with b1=x,a0=1−xb_1=x,a_0=1-xb1​=x,a0​=1−x.

This now gives us the output of a first-order low-pass filter with time constant ddd:

yn=(1−e−1/d)xn+e−1/dyn−1y_n = (1-e^{-1/d})x_n + e^{-1/d}y_{n-1}yn​=(1−e−1/d)xn​+e−1/dyn−1​

The simplest second-order system​

In order to easily add a single degree of complexity, namely to create a system whose output depends on the second-order time derivatives of the input, as can simply chain two first-order low-pass filters together lpf_2 -> lpf_1 ( x_n ). Doing this yields the following second-order transfer function:

x(ω)=a01−b1ω⋅c01−d1ω=a0c01−(b1+d1)ω+b1d1ω2x(\omega) = \frac{a_0}{1-b_1\omega}\cdot\frac{c_0}{1-d_1\omega} = \frac{a_0c_0}{1-(b_1+d_1)\omega+b_1d_1\omega^2}x(ω)=1−b1​ωa0​​⋅1−d1​ωc0​​=1−(b1​+d1​)ω+b1​d1​ω2a0​c0​​

where ccc, and ddd are the recursion coefficients of the second filter. We can now write the system output based on these sets of coefficients just as before:

yn=(1−e−1/d1)(1−e−1/d2)xn+(e−1/d1+e−1/d2)yn−1−e−1/d1−1/d2yn−2y_n = (1-e^{-1/d_1})(1-e^{-1/d_2})x_n + (e^{-1/d_1}+e^{-1/d_2})y_{n-1} - e^{-1/d_1 - 1/d_2}y_{n-2}yn​=(1−e−1/d1​)(1−e−1/d2​)xn​+(e−1/d1​+e−1/d2​)yn−1​−e−1/d1​−1/d2​yn−2​

where d1d_1d1​, and d2d_2d2​ are the time constants of each of the filters respectively.
