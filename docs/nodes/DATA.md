Welcome to the Data Nodes page!

Here you can find a list of all the Flojoy nodes that you can use to generate certain outputs, for instance by utilizing Artificial Intelligence (AI) and Machine Learning (ML) or as part of simulations, as well as to visualize your data.

You can also find all of these nodes in the sidebar.
